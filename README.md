**🔑 Key Features**

- 🔐 **QR Code Generation** — Unique pass generation for each student exit
- 🛡️ **Warden Dashboard** — Real-time monitoring & approval workflows

```
QR Code Generated ← Database Record ← Approval Status

Warden Dashboard (Real-time tracking)
```
